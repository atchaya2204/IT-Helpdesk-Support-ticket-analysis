# IT-Helpdesk-Support-ticket-analysis
This project focuses on analyzing IT support tickets to improve operational efficiency and service quality. A dataset containing ticket details such as priority, category, assigned personnel, resolution time, and SLA status was used to build an interactive dashboard in Power BI.

Key features of the dashboard include:

KPI tracking for total tickets, resolved tickets, open tickets, and SLA breaches

Time-based analysis to identify ticket trends and peak periods

Category-wise and priority-wise breakdown to detect major issue areas

Employee performance analysis based on resolution time

SLA monitoring to identify delays and improve service levels

DAX functions were used to create calculated measures for business insights, and slicers were implemented to enable dynamic filtering. The dashboard helps stakeholders quickly identify problem areas, optimize resource allocation, and improve response times.
